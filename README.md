# fr_test_task

### Установка 
1. copy vars from env template to .env
```console
foo@bar:~$ cp .env.template .env
```
2. run docker
```console
foo@bar:~$ docker-compose up --build
```

### Все методы апи можно найти на [0.0.0.0:8000/docs](http://0.0.0.0:8000/docs/)
